**Dungeon gates**


За основу взята игра Пакман. Но здесь персонаж не просто ходит по лабиринту с монстрами, а проходит уровни и получает ключи, чтобы открыть двери, ведущие к выходу из лабиринта. Всего три уровня: крестики-нолики, 6561 и змейка.

**Крестики-нолики**


Мы сделали 5 различных вариаций крестиков-ноликов. Например, вариант с гравитацией и поворотом доски. Программа случайно выбирает 1 из 5 вариантов, и пользователь играет против бота.
